# 데모 스킬 지침

이것은 데모 스킬입니다. 나만의 스킬을 만들기 위한 템플릿 역할을 합니다.

## 구조
- **YAML Frontmatter**: 맨 위에 있는 블록으로 `name`(이름)과 `description`(설명)을 포함합니다.
- **Instructions (지침)**: 아래의 내용은 이 스킬이 활성 상태일 때 제가 따르게 될 지침입니다.

## 기능
이 스킬이 호출되면 저는 다음과 같은 작업을 수행할 수 있습니다:
1.  여기에 정의된 특정 코딩 스타일을 따릅니다.
2.  이 폴더에 있는 특정 템플릿을 사용합니다 (추가된 경우).
3.  특정 스크립트를 실행합니다 (추가된 경우).

## 예제 작업
저에게 "데모 스킬을 시연해줘"라고 요청하시면, 저는 다음과 같이 출력할 것입니다:
"안녕하세요! 데모 스킬이 올바르게 작동하고 있습니다."
